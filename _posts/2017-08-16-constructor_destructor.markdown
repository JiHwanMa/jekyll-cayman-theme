# 생성자란?
~~~
특징1. : 클래스의 이름과 함수의 이름이 동일하다.
특징2. : 반환형이 선언되어있지 않으며, 실제로 반환하지 않는다. 
->객체 생성 시 딱 1회만 호출됨.
->오버로딩, 디폴트값 설정 가능.
~~~
# 소멸자란?
~~~
특징1. : 클래스의 이름 앞에 ~가 붙은 형태.
특징2. : 반환형이 없으며 실제로 반환하지 않는다.
특징3. : 매개변수는 void 형으로 선언되어야 하기 때문에 오버로딩,디폴트 값 설정이 둘다 불가하다.
~~~
실습코드.
~~~c
#include<iostream>
#include<cstdlib>
using namespace std;
~~~
